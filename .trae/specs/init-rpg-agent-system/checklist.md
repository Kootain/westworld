# Checklist

- [ ] 项目目录结构划分清晰，代码符合单一职责与高度抽象原则（Clean Code）
- [ ] 输入输出模块完成抽象（插件化 I/O），并支持通过 Console 正常收发信息
- [ ] 感官记忆能够正确记录当前的地点、天气、NPC 数值与刚发生的事件
- [ ] 短期记忆 (STM) 的衰减算法（基于重要性和时间）能正确计算，并在超过阈值时逐出过期记忆
- [ ] 长期记忆 (LTM) 固化机制可以成功提取 STM 的核心事实和关系变化并持久化
- [ ] 处于“不可交互”状态（如睡觉）的智能体，流量会被 Box 拦截并返回正确的系统提示
- [ ] Box 能够成功屏蔽智能体的内部思维 (SoT)，向外部仅输出对话 (Speech) 和动作 (Action)
- [ ] 智能体 Sub-Graph 能按顺序正确执行 Perceive -> Recall -> SoT -> Act
- [ ] 全局 LangGraph 能够正常运转，形成完整的对话闭环
- [ ] 后台 Tick 系统能够正常模拟时间流逝并自动向系统注入环境事件
- [ ] 当状态发生重大变更时，系统能成功触发虚拟朋友圈生成逻辑（提取 LTM 并生成图像 Prompt 及匹配文案）
